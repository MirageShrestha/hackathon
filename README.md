# Sojourner — Travel Companion for Nepal 🏔️

A mobile-first travel planning web app built with React, TypeScript, Tailwind CSS, and shadcn/ui. Sojourner helps users discover vacation experiences, explore destinations, manage trip planning, and access service options across Nepal.

---

## 🎯 Problem

Travelers planning trips to Nepal face several significant challenges:

- **Information Overload**: Scattered information across multiple platforms makes it difficult to find reliable, comprehensive trip planning resources
- **Budget Uncertainty**: Hidden costs and lack of transparent pricing for trekking, accommodation, and transportation
- **Planning Complexity**: Coordinating itineraries, guides, permits, and logistics without local expertise
- **Safety Concerns**: Limited access to verified guides, updated trail conditions, and emergency contacts
- **Community Disconnect**: Lack of platforms to share experiences, connect with fellow travelers, and learn from local experts
- **Service Discovery**: Difficulty finding trustworthy local services, accommodations, and transportation options

---

## 💡 Solution

Sojourner provides a comprehensive, mobile-first platform that centralizes Nepal's travel ecosystem:

- **Unified Discovery**: Single platform for reels, itineraries, services, and personal trip management
- **Transparent Planning**: Detailed budget breakdowns, day-by-day itineraries, and cost tracking
- **Community-Driven**: User-generated content, expert guides, and traveler experiences
- **Local Expertise**: Direct access to verified guides, porters, and local service providers
- **Personal Journey**: Save plans, track expenses, and build a personal travel journal
- **Mobile-First Design**: Optimized for on-the-go planning and exploration

### Target Stakeholders

**Content Creators (Primary Focus)**
- Micro-influencers (10K-100K followers) create travel reels
- Earn 20% commission on every booking their content generates
- Get paid via eSewa/Khalti on weekly basis
- Access to analytics dashboard tracking real-time earnings
- Incentive: Top creators earn ₹20K-50K/month

**Travelers**
- Browse free reel feed for trip inspiration
- Book guides directly through creator recommendations
- Get transparent pricing with no hidden costs
- Save plans to personal trek profile
- Optional premium features: ₹200/month

**Guides & Service Providers**
- List services for free, pay only when they get bookings
- Keep 80-85% of booking value (vs. 40-50% with tour operators)
- Build personal brand with reviews and ratings
- Optional premium badge for visibility: ₹500/month

**Hotels & Lodges**
- Get bookings from platform for 10% commission
- Free marketing to verified travelers
- Direct access to content creators for partnerships

### Go-To-Market Strategy

**Phase 1: Creator Acquisition (Months 1-2)**
- Target 50 micro-influencers on YouTube/Instagram/TikTok
- Offer 20% commissions (higher than standard 15%)
- DM outreach with affiliate link
- Goal: 10 creators onboarded

**Phase 2: Content Distribution (Months 3-4)**
- Creators post travel reels featuring guides/lodges
- Each reel includes booking link in bio
- Travelers book through creator link → commission earned
- Goal: 30 active creators, 200+ bookings/month

**Phase 3: Scale (Months 5-12)**
- Creator leaderboard (earnings transparency)
- Monthly creator challenges with prize pool
- Sponsor micro-creators with free guide passes
- Expand to 60+ creators, ₹300K+ monthly revenue

### Unit Economics

**Per Booking (Average ₹2,000 guide day rate):**
- Sojourner commission: ₹300 (15%)
- Creator share: ₹400 (20%)
- Platform cost per booking: ₹50
- Net profit: ₹250 per booking

**Breakeven:** 160 bookings/month = ₹40,000 (covers ₹40K monthly operations)

**Growth Target:** 500+ bookings/month by Month 12 = ₹125,000 monthly profit

- **Budget Optimization**: AI algorithms analyze historical data to suggest cost-saving opportunities

- **Risk Assessment**: AI evaluation of trail conditions, weather patterns, and safety factors
- **Emergency Response**: Intelligent routing to nearest medical facilities and emergency contacts

## ⚡ Getting Started

### Requirements

- Node.js 18 or newer
- npm or yarn package manager

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

Open `http://localhost:5173` in your browser.

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

---

## 🧪 Testing

Run unit tests with:

```bash
npm run test
```

Run watch mode with:

```bash
npm run test:watch
```

## 💡 Development Notes

- Google Maps integration is handled in `ReelMapSection.tsx`
- Mock data is defined in `src/components/trekking/data.ts`
- Main application state is managed in `src/pages/Index.tsx`
- Browser routing is configured in `src/App.tsx`
- Local storage is used for data persistence across sessions

---

## 🧭 Development Tips

- Update reel data in `src/components/trekking/data.ts`
- Add new UI components under `src/components/`
- Maintain consistent styling with Tailwind CSS utilities
- Use Framer Motion for smooth animations and transitions
- Follow TypeScript best practices for type safety
